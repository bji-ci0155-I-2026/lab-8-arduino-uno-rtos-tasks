# Laboratorio #8: Programación de tareas usando sistemas operativos de tiempo real

## Objetivo general

Programar mediante el uso de un sistema operativo de tiempo real distintas tareas en un sistema embebido (Circuit Playground Classic o Arduino Uno R3).

## Parte 1: FreeRTOS

- Investigue sobre FreeRTOS y su uso en el desarrollo de sistemas empotrados de tiempo real.
- Investigue sobre la programación y calendarización de tareas usando FreeRTOS.

## Parte 2: Programación

- Investigue sobre la biblioteca `Arduino_FreeRTOS_Library` para dispositivos Arduino con ATmega.
- Usando la biblioteca `Arduino_FreeRTOS_Library` y una de las plataformas disponibles en el curso (Circuit Playground Classic o Arduino Uno R3) programe:
  - Varias tareas con distintas prioridades.
  - Una tarea periódica.
- Por cada tarea programada verifique el funcionamiento.

## Entregables

- Código fuente.
- Evidencia de pruebas (capturas o descripción de resultados).
- Documento con resultados de las investigaciones solicitadas y análisis de la programación de tareas usando FreeRTOS.
